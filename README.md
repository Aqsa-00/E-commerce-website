# ECOBUY - Fashion Store

ECOBUY is a modern, responsive fashion e-commerce website showcasing featured products, new arrivals, and seasonal sales. This project focuses on a clean, visually appealing desktop design with smooth navigation and structured layout. 

## Features
- Sticky header with navigation menu and brand logo
- Hero section with promotional offers
- Featured services section (Free Shipping, Online Order, etc.)
- Product sections with ratings and add-to-cart buttons
- Banner sections for seasonal offers and promotions
- Newsletter signup section
- Footer with contact information, social media links, and account navigation
- Fully responsive layout (desktop-focused)

## Technologies Used
- HTML5
- CSS3 (with CSS variables for theme colors)
- Google Fonts (Poppins)
- FontAwesome icons
